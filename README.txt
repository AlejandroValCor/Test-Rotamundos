Instrucciones de ejecución:

1.- Para ejectuar el programa se puede descargar el archivo .py en el directorio que guste. 
2.- Posteriormente es necesario abrir un símbolo de sistema y navegar al directorio donde se descargo el .py. 
3.- Una vez dentro del mismo directorio donde se descargo el .py, ingresar la siguiente línea de comando en el símbolo de sistema: python "Test Rotamundos.py"

El programa ya incluye todos los casos de prueba incluyendo a cada alumno y un caso extra en el caso de no encontrar al alumno solicitado.
Al igual que los casos de prueba para el promedio total del grupo y para indicar el alumno con el mayor promedio dentro del grupo.
